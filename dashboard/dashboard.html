🎭 Modo de conversación del bot

Define el estilo con el que el bot le habla a los clientes. El cambio aplica en menos de 60 segundos, sin reiniciar el bot.

👔 Formal

Activo

Trato de usted, lenguaje corporativo. Para clientes mayores o B2B.

"Bienvenido a Julio Cepeda Jugueterias. Es un gusto atenderle."

😎 Casual

Activo

Directo y breve. Mensajes cortos para respuestas rapidas.

"Hola! Soy el bot de Julio Cepeda. Que necesitas?"

🧸 Amigable

Activo

Calido y con emojis. Tono por defecto, equilibrado para todo publico.

"Hola! Bienvenido a Julio Cepeda Jugueterias 🎉"

🎯 Ventas (23-40)

Activo

Beneficio primero, urgencia honesta. Optimizado para conversion.

"Llegaste a Julio Cepeda, 600 juguetes con entrega hoy mismo."

🎨 Apariencia del panel

Elige el tema visual del panel. Es una preferencia personal de tu navegador, no afecta al bot ni a otros usuarios.

🌿 Claro suave

Activo

Tonos fríos y suaves. Recomendado para usarlo varias horas.

🌅 Claro cálido

Activo

Tonos cálidos tipo arena. Misma legibilidad, ambiente más cercano.

🌙 Modo oscuro

Activo

Fondo oscuro, ideal con poca luz o turnos nocturnos.

👑 APIs reales (Prime)

Requiere credenciales de usuario prime (distintas a las de este panel). Enciende el cobro real y el envío real con Estafeta — hoy ambos están simulados.

💳 Pago real (PayPal)

Cobrar de verdad en vez de simular el link de pago

🚚 Estafeta real

Generar guías reales en vez de simularlas

💲 Editar costo de envío

Corrige el costo de envío de un pedido ya creado (ej. Estafeta cotizó distinto a la simulación). Recalcula el total del pedido.

🧪 Reset betatestor

Limpia datos de prueba de un número específico.

Presiona el botón para verificar
